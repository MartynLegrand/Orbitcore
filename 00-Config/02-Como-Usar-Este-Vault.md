# Como Usar Este Vault

## Objetivo
- Navegar todo o conhecimento do ecossistema Orbit Corp/OBM em notas interligadas.
- Acelerar criação de módulos e apps com padrões claros e receitas práticas.

## Navegação
- Use os índices das pastas e os wikilinks (ex.: `[[Glossario-Tecnico/JWT]]`).
- Comece pela [[01-Visao-Geral/01-Resumo-Estrategico]] e [[01-Visao-Geral/03-Modulos-OBM-Visao-Geral]].
- Consulte [[00-Config/03-RULES-GLOBAL]] antes de criar ou alterar qualquer módulo.

## Convenções
- Prefixo de módulos: `obm-*`.
- Use placeholders de apps: ver [[00-Config/00-Placeholders]].
- Todo termo técnico deve ter uma nota em `06-Glossario-Tecnico/`.
