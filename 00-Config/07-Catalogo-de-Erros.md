# Catálogo de Erros Orbit OBM

## AUTH
- AUTH_INVALID_CREDENTIALS → 401
- AUTH_TOKEN_EXPIRED → 401
- AUTH_FORBIDDEN → 403
- AUTH_RATE_LIMITED → 429

## PAYMENT
- PAYMENT_METHOD_INVALID → 422
- PAYMENT_INSUFFICIENT_FUNDS → 402
- PAYMENT_WEBHOOK_INVALID_SIGNATURE → 400
- PAYMENT_RETRY_LATER → 503

## NOTIFY
- NOTIFY_CHANNEL_UNAVAILABLE → 503
- NOTIFY_TEMPLATE_MISSING → 404
- NOTIFY_RATE_LIMITED → 429

## SEARCH
- SEARCH_QUERY_INVALID → 422
- SEARCH_INDEX_MISSING → 404

## FILES
- FILES_UPLOAD_INVALID → 422
- FILES_NOT_FOUND → 404
- FILES_ACCESS_DENIED → 403

## COMMON
- VALIDATION_ERROR → 422
- NOT_FOUND → 404
- RATE_LIMITED → 429
- INTERNAL_ERROR → 500

> Alinhar códigos nas respostas conforme [[00-Config/05-Protocolo-Orbit-OBM]].
