# WebAssembly (WASM) ⚙️

**O que é:**
Formato binário portátil para executar módulos em sandbox.

**Por que importa aqui:**
Pluginização de componentes de alta performance (Rust → WASM).

**Exemplo:**
- Carregar módulo WASM no Node/Browser para processamento.
